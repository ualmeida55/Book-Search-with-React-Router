# Book Search App

Web App to search books using an external API.  

Deployed using Vercel: [Live Demo](https://book-search-with-react-router-oqdw.vercel.app)  

## Built with  
- React
- React Router  
- Bootstrap  
- HTML  
- CSS    
- Vite

## API
Provided by [OpenLibrary](https://openlibrary.org/developers/api)
